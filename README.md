# Практическая работа №2
### Вариант 11: Персональный компьютер

## Инструкция по сборке
1. Распаковать проект
2. Открыть в терминале папку с проектом
3. Для сборки
   ```
   mvn compile
   ```
4. Для запуска
   ```
   mvn exec:java
   ```
_Для сборки необходим [Maven](https://maven.apache.org/download.cgi)_
